# diy-continuity-tester

Why I made this:

I needed to learn to solder so I made this project. It runs 3.3v through one lead and the other lead expects it, and that activates a buzzer and an LED to signal electrical connection.

The solder isn't the best, but I learned how to use my iron so I consider it a success.
